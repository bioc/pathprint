Description
Annotates an expression array with entrez gene IDs. according a custom mappings, averaging to resolve redundancies

Usage
customCDFAnn(data, ann)

Arguments
data	expression dataframe
ann	annotation dataframe

Details


Value
Returns a dataframe with rownames as unique entrez IDs
